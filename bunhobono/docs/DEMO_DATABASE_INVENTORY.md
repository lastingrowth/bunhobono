# 데모 DB 데이터 구성표

기준 파일: `db/demo_dummy_full.sql`

> 아래 수치는 빈 테이블에 SQL 전체가 오류 없이 실행된 직후의 예상 건수입니다.
> 차량 등록 수, 차량번호 수, 입출차 기록 수는 서로 다른 개념이므로 구분해 표시합니다.

## 1. 차량 등록 (`vehicle_car`)

| 구분 | 승인 완료 | 승인 대기 | 합계 |
|---|---:|---:|---:|
| 입주민 차량 (`normal`) | 55 | 0 | 55 |
| 방문 차량 (`visit`) | 11 | 3 | 14 |
| 합계 | 66 | 3 | **69** |

입주민 승인 완료 55대:

- 이미지 연계 시연 차량 48대
- 지정 필수 차량 7대

방문 승인 완료 11대:

- 이미지 연계 시연 차량 9대
- 지정 필수 차량 2대

지정 필수 차량 9대는 등록만 되어 있고 입출차 기록에는 사용하지 않습니다.

## 2. 실제 입출차에 사용한 고유 차량번호

| 구분 | 고유 차량번호 |
|---|---:|
| 입주민 | 48대 |
| 방문 | 8대 |
| 미등록 | 4대 |
| 합계 | **60대** |

같은 차량번호가 과거 출차 완료 기록과 현재 주차 기록에 다시 등장할 수 있으므로,
고유 차량번호 60대가 총 108개의 카로그를 만듭니다.

등록되어 있지만 카로그에 사용하지 않은 차량은 13대입니다.

- 지정 필수 차량 9대
- 승인된 방문 차량 중 기록 미사용 1대
- 방문 승인 대기 3대

| 미등록 | 4 | 6.7% |

## 4. 출차 완료

| 구분 | 기록 수 |
|---|---:|
| 입주민 | 40 |
| 방문 | 4 |
| 미등록 | 4 |
| 합계 | **48** |

게이트마다 입주민 10건, 방문 1건, 미등록 1건으로 동일하게 배치됩니다.

## 5. 활성 카로그와 카메라 데이터

| 테이블/자료 | 건수 |
|---|---:|
| `car_log` 전체 | **108** |
| 현재 주차 카로그 | 60 |
| 출차 완료 카로그 | 48 |
| `camera_data` 전체 | **156** |
| 입차 촬영 | 108 |
| 출차 촬영 | 48 |

카로그마다 입차 촬영 1건이 있고, 출차 완료 카로그 48건에는 출차 촬영도
1건씩 있어서 카메라 데이터가 카로그보다 48건 많습니다.

카메라 촬영 분류:

| 구분 | 입차 | 출차 | 합계 |
|---|---:|---:|---:|
| 입주민 | 88 | 40 | 128 |
| 방문 | 12 | 4 | 16 |
| 미등록 | 8 | 4 | 12 |
| 합계 | **108** | **48** | **156** |

이미지 파일은 원본 100개와 크롭 100개가 백엔드에 있으며, SQL에서 차량번호와
확인된 71개 이미지 매핑을 입출차 기록에 반복 연결합니다.

## 6. 지난 기록 (`trash_bin`)

| 유형 | 건수 |
|---|---:|
| `CAR_LOG` | **3,468** |
| `CAMERA_DATA` | 12 |
| `NOTICE` | 12 |
| 전체 | **3,492** |

`CAR_LOG` 지난 기록 3,468건 구성:

| 목적 | 계산 | 건수 |
|---|---|---:|
| 시간대 그래프 `HOURLY` | 22일 × 하루 100개 | 2,200 |
| 최근 입차·평균시간 | 22일 × 하루 44개 | 968 |
| 2026년 1~6월 연간 통계 | 월별 합계 | 288 |
| 일반 지난 카로그 | 페이지네이션용 | 12 |
| 합계 |  | **3,468** |

시간대 그래프의 2,200건은 실제 서로 다른 차량 2,200대가 아니라,
22일 동안의 시간대별 점유 상태를 만들기 위한 통계 스냅샷입니다.

최근 22일 입차·평균시간 자료 968건:

## 7. 알림

활성 `notice`:

| 상태 | 건수 |
|---|---:|
| 미확인 (`Unresolved`) | 2 |
| 확인 (`Checked`) | 1 |
| 처리 완료 (`Resolved`) | 3 |
| 합계 | **6** |

차량 종류는 방문 3건, 미등록 3건입니다.

방문 차량 승인 대기는 별도로 `vehicle_car`에 3건 있습니다.
